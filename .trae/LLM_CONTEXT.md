# Oil Wars — планшет фракции (вёрстка A4 для печати)

Этот проект — одностраничная веб-вёрстка планшета фракции для настольной игры. Макет ориентирован на печать в формате A4 (альбомная ориентация), с интерактивным редактированием содержимого, локальным сохранением в браузере, экспортом/импортом JSON и поддержкой верстки «задней стороны» планшета.

## Назначение
- Быстро собрать планшет фракции: название, логотип, свойства юнитов, таблица отрядов, памятки.
- Печатать без артефактов: фон, шрифты, линии и SVG-оверлей связей учитывают печать.
- Обмениваться данными через экспорт/импорт JSON (включая изображения как data URL).

## Как запустить
- Откройте файл Template.html в браузере (двойной клик или через file://).
- Для изолированного рабочего пространства используйте параметр URL id:
  - Пример: Template.html?id=MoonSystems — все сохранения попадут в отдельный неймспейс.

## UI и поведение
- Название фракции: редактируемый блок; авто-подбор кегля под доступную ширину (бинпоиск).
- Логотип фракции: дропзона (клик, перетаскивание, paste ссылки/data URL).
- Панель подготовки: очередь, запас нефти, регион, заметки, блок «Создание боевого робота» (заголовок + упорядоченный список). Вставки в список санитизируются и дробятся по строкам.
- Панель свойств юнитов: карточки свойств, каждое с title/note/desc. Перетаскивание только за «⋮⋮» у выбранного блока. Высота и top абсолютны и сохраняются.
- Таблица отрядов:
  - 1-я колонка: иконка (дропзона) + подпись юнита (contenteditable).
  - Далее 3 числовых столбца (класс unit-stat, единый шрифт для стабильности).
  - Разделители групп: «горизонтальная линия» между блоками. Тип отряда (вертикальный лейбл) задаётся в первой строке группы и тянется через группу (rowSpan).
  - Выбор строки кликом; Delete удаляет выбранную строку (кроме случаев ввода), Escape снимает выделение.
- Режим «Связь»: кнопка «↗ Связь» → клик по свойству → клик по подписи юнита. Рисуется полилиния с «входом» в подпись. Позиции обновляются при resize/scroll/drag, сохраняются через id.
- Задняя сторона: верх — квадратная иллюстрация слева и художественное описание справа; низ — гайд «Как играть…». Вся разметка внутри «safe area» с полями 20 мм; избыточный текст прокручивается внутри областей.

## Сохранение и неймспейсы
- Неймспейс ns берётся из ?id=…; если нет — из имени файла; иначе 'default' ([app.js:1–4](file:///c:/YandexDisk/Oil%20Wars/Templates/app.js#L1-L4)).
- Ключи localStorage формируются от ns:
  - faction-name:${ns}
  - unit-table:${ns}
  - unit-props:${ns}
  - unit-links:${ns}
  - faction-info:${ns}
  - back-info:${ns}
- Все вводимые изменения дебаунсятся и сохраняются автоматически; при несохранённых изменениях показывается beforeunload-предупреждение, которое подавляется после экспорта.

## Экспорт / импорт
- Экспорт собирает всё состояние в один JSON и скачивает файл (включая изображения как data URL):
  - [storage.js:94–157](file:///c:/YandexDisk/Oil%20Wars/Templates/storage.js#L94-L157)
- Импорт принимает либо «плоский» массив rows, либо полный объект; применяет: таблицу, название фракции, свойства, связи, подготовку и задник:
  - [storage.js:159–223](file:///c:/YandexDisk/Oil%20Wars/Templates/storage.js#L159-L223)

Мини-спецификация экспортируемого JSON:
```json
{
  "factionName": "string",
  "rows": [
    { "type": "separator" },
    {
      "type": "data",
      "imgSrc": "data:image/... | http(s)://...",
      "caption": "string",
      "cells": ["", "", ""],           // три числовых/текстовых значения
      "unitId": "uuid-like",
      "groupLabel": "string?"          // только на первой строке каждой группы
    }
  ],
  "unitProps": [
    { "id": "uuid-like", "title": "string", "note": "html", "desc": "html", "y": number }
  ],
  "unitLinks": [
    { "propId": "uuid-like", "unitId": "uuid-like" }
  ],
  "factionInfo": {
    "queue": "string", "oil": "string", "region": "string",
    "notes": "html", "robotTitle": "string", "robotSteps": "html",
    "logoSrc": "data:image/..."
  },
  "backInfo": {
    "desc": "html",
    "guideTitle": "string",
    "guideText": "html",
    "imgSrc": "data:image/..."
  }
}
```

## Печать
- Формат: @page A4 landscape; нулевые поля, фон страницы — background.png.
- В печати скрываются: тулбар, рамки дропзон, хэндлы; усиливается линия-разделитель.
- SVG-оверлей связей масштабируется под страницу и печатается.
