# Шаблон для жетонов фабрик Spec

## Why
Необходим статический HTML/CSS шаблон для качественной печати жетонов фабрик (размером 50×50 мм) на листах формата A4. Шаблон должен быть готов к печати без дополнительных настроек браузера.

## What Changes
- Создание файла `Factory.html`
- Создание директории `Factory/` с файлом `factory.css`
- Создание заглушек для изображений `factory.png` и `Factory2.png`
- Реализация HTML/CSS сетки для 4×5 жетонов (размером 50×50 мм) с центрированием
- Добавление внешних вылетов под обрез (bleed) и меток реза (crop marks) только для режима печати

## Impact
- Affected specs: Печать компонентов настольной игры
- Affected code: `Factory.html`, `Factory/factory.css`

## ADDED Requirements
### Requirement: Печать на A4
Шаблон должен строго соответствовать формату A4 (210×297 мм) в портретной ориентации без полей браузера, без использования JS.
#### Scenario: Success case
- **WHEN** пользователь открывает страницу и нажимает печать
- **THEN** браузер предлагает печать в масштабе 100% на формате A4 без лишних отступов и полей

### Requirement: Сетка жетонов
Жетоны размером 50×50 мм размещаются в сетке 4×5 (20 штук на страницу).
#### Scenario: Success case
- **WHEN** страница рендерится
- **THEN** жетоны выровнены строго по центру листа с добавлением внешних вылетов (bleed) и меток реза (светло-серых)

### Requirement: Внешние вылеты и метки
Вылеты (bleed) должны быть только вокруг всего блока жетонов (не между ними), а метки реза должны быть видны только в режиме печати.
#### Scenario: Success case
- **WHEN** пользователь просматривает предпросмотр печати
- **THEN** по углам каждой карты и между картами отображаются светло-серые кресты меток реза, а вокруг всей группы добавлен внешний отступ цвета `#191b1c`
