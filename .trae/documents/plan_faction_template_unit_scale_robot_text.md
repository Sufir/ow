# План доработок: масштаб unit-icon + форматирование robot-panel

## Summary
- Внести ровно две доработки в шаблон фракции: (1) масштабирование иконок юнитов с кнопками `+/-` при выборе строки таблицы, (2) форматирование выделенного текста в `robot-panel` по образцу механики из `mini-cards/cards-landscape-sixes.js`, с аккуратной адаптацией и минимальными изменениями.
- Обеспечить полную совместимость с текущими сценариями сохранения/загрузки: `localStorage`, экспорт JSON, импорт JSON.

## Current State Analysis
- `Template.html` содержит тулбар без кнопок масштабирования юнит-иконок и без контролов форматирования для `robot-panel`.
- `faction-card/app.js`:
  - Выбор строки уже реализован через `selectRow(tr)` и класс `.row-selected`.
  - Инициализация/рендер иконок сделаны через `buildUnitCell`, `renderTable`, `ensureDropzone`.
  - Есть сохранение таблицы через `saveTable()` и полная цепочка импорта/экспорта через `StorageAPI`.
  - Для `robot-panel` есть редактирование и сохранение, но текущая санитизация `sanitizeRobotSteps` удаляет inline-стили (`removeAttribute('style')` у `li`), что несовместимо с хранением форматирования.
- `faction-card/storage.js`:
  - В `buildRowsFromTbody` сериализуются `imgSrc`, `caption`, `cells`, `unitId`, `groupLabel`.
  - Модель строки таблицы пока не содержит параметр масштаба иконки.
  - Экспорт/импорт JSON уже берет данные таблицы из `tableStorageKey`; расширение структуры строк автоматически попадет и в JSON.
- `faction-card/style.css`:
  - Размер иконок фиксирован через `.unit-image { width: 100%; height: 12mm; }`.
  - Нет UI-стилей для скрываемых кнопок `+/-` масштабирования.
  - Есть стили `robot-panel`, но нет плавающих контролов форматирования.
- Эталонная логика форматирования в `mini-cards/cards-landscape-sixes.js`:
  - Показ/скрытие контролов в зависимости от активного редактируемого поля.
  - Команды `fontName`, `bold`, `italic`, изменение размера шрифта шагом `0.5`.
  - Сохранение HTML после санитизации.

## Assumptions & Decisions
- Доработки ограничиваются только двумя заявленными пунктами; новые функции вне запроса не добавляются.
- Кнопки масштаба видны/активны только при выборе строки данных (не комментарий/разделитель).
- Шаг масштаба иконки: `0.05`; диапазон: `0.5..1.8` (выбран как безопасный для текущей верстки).
- Шаг изменения шрифта в `robot-panel`: `0.5`, как в эталонной реализации.
- Форматирование применяется только к выделенному тексту в `.robot-title` и `.robot-steps`.
- Для печати новые UI-контролы скрываются.
- Совместимость со старыми сохранениями обязательна: отсутствие новых полей трактуется как дефолт (`unitScale=1`, без форматирования).
